CPU Benchmark:1) Download the file MyCPUBench.c (Source code) onto the Linux machine.2) Download two bash script files CPUMaster.sh and CPUSlave.sh onto the Linux machine.3) Run the CPUMaster.sh bash script as below.     	    ./CPUMaster.sh		4) The CPUMaster.sh will compile the code, run different test cases and will call CPUSlave.sh to submit jobs on different   compute nodes.5) The results of all the tests will be written in different files using the format "cpu_$operation_$threadcount.out.dat".6) The output will contain information in below format:		------------------------------------------------------------------- Workload   Concurrency  MyCPUBenchValue TheoreticalValue Efficiency  DP            1           2.217331 -------------------------------------------------------------------
