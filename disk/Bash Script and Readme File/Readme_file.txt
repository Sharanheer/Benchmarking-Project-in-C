 Disk Benchmark:1) Download the file MyDiskBench.c (Source code) onto the Linux machine.2) Download two bash script files DiskMaster.sh and DiskSlave.sh onto the Linux machine.3) Run the RAMMaster.sh bash script as below.     	    ./DiskMaster.sh		4) The DiskMaster.sh will compile the code, run different test cases and will call DiskSlave.sh to submit jobs on different compute nodes.5) The results of all the tests will be written in different files using the format 		"disk_$operation_$block_size_$threadcount.out.dat".6) The output will contain information in below format:		----------------------------------------------------------------------------Workload Concurrency BlockSize MyDiskBenchValue TheoreticalOutput Efficiency  RS         1          1MB         137.18 ----------------------------------------------------------------------------
